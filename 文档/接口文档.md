## POST 搜索依赖项（新）

POST /api/internal/browse/components

搜索依赖

> Body 请求参数

```json
{
    "page": 0,
    "size": 10,
    "searchTerm": "lombok",
    "filter": []
}
```

### 请求参数

|名称|位置|类型|必选|说明|
|---|---|---|---|---|
|body|body|object| 是 |none|
|» page|body|integer| 是 |第几页|
|» size|body|integer| 是 |返回项数|
|» searchTerm|body|string| 是 |搜索条目|
|» filter|body|[string]| 是 |none|
