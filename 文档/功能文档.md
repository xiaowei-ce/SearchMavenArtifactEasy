

## 项目：Maven Artifact Searcher (MAS) 插件开发规格书

### 1. 技术栈与环境

- **开发语言**: Java 17+
- **SDK**: IntelliJ Platform Plugin SDK
- **外部依赖**: `Gson` (用于解析 JSON, IDEA 自带)

------

### 2. UI 布局规格 (纯 Java 代码实现)

对话框类继承 `DialogWrapper`，并在 `createCenterPanel()` 中按以下结构布局：

- **面板结构**: `JPanel` (BorderLayout)
  - **North**: `JPanel` (FlowLayout, Left)
    - `JBTextField` (变量名: `searchField`): 宽度 400，设置空提示。
    - `JButton` (变量名: `searchBtn`): 文本 "Search"。
  - **Center**: `JBScrollPane` 包裹 `JBTable`
    - `DefaultTableModel`: 列名 `["Group ID", "Artifact ID", "Latest Version"]`。
  - **South**: `JPanel` (Vertical BoxLayout)
    - **Version Row**: `FlowLayout: Left` -> `JLabel("Version: ")` + `ComboBox<String>` (变量名: `versionCombo`)。
    - **Preview Row**: `BorderLayout` -> `JTextArea` (变量名: `xmlPreview`): 设置背景色为 `0x2B2B2B`，文字为绿色或灰色，只读。
    - **Action Row**: `FlowLayout: Right` -> `JButton` (变量名: `copyBtn`) + `JButton` (变量名: `addToPomBtn`)。

------

### 3. 核心功能逻辑

#### **功能 A：搜索与异步请求**

1. **监听**: 为 `searchBtn` 和 `searchField` 的回车键绑定事件。
2. **异步执行**: 使用 `new Task.Backgroundable(project, "Searching...")` 发起请求。
3. **API**: 见接口文档
4. **回调**: 请求结束后，使用 `SwingUtilities.invokeLater` 刷新 `JBTable`。

#### **功能 B：版本联动**

1. **监听**: 为 `JBTable` 的选中模型添加 `ListSelectionListener`。
2. **触发**: 选中行时，异步请求该 GAV 的所有版本：见接口文档
3. **更新**: `versionCombo` 的第一项固定为 `"[No Version / Omit]"`，后续为搜索到的版本列表。

#### **功能 C：智能预览**

1. **逻辑**: 监听表格行和版本下拉框的切换。

2. **模板**: 根据当前选择动态生成字符串：

   XML

   ```
   <dependency>
       <groupId>[G]</groupId>
       <artifactId>[A]</artifactId>
       <version>[V]</version> 
   </dependency>
   ```

#### **功能 D：写入 pom.xml**

1. **权限控制**: 初始化时判断当前文件是否为 `pom.xml`，若不是，`addToPomBtn.setEnabled(false)`。
2. **PSI 操作**:
   - 使用 `PsiManager.getInstance(project).findFile(currentFile)` 获取 `XmlFile`。
   - 找到根标签 `<project>` 下的 `<dependencies>`。
   - **如果缺失**: 调用 `projectTag.createChildTag("dependencies", null, null, false)` 创建。
   - **插入**: 调用 `dependenciesTag.addSubTag(newTag, false)`。
3. **安全执行**: 整个插入逻辑必须包裹在 `WriteCommandAction.runWriteCommandAction` 中。

------

### 4. 异常处理规格

- **空结果**: 若搜索结果为空，在 `JBTable` 中显示一行 "No artifacts found"。
- **网络断开**: 捕获 `IOException`，使用 `Messages.showErrorDialog` 或 `Notification` 提醒用户。
- **并发保护**: 在搜索进行中，禁用 `searchBtn` 以防重复点击。
